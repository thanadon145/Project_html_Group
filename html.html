HTML Tutorial

HTML HOME

หน้านี้เป็นจุดเริ่มต้นสำหรับผู้ที่สนใจศึกษาเกี่ยวกับ HTML (HyperText Markup Language) ซึ่งเป็นภาษาหลักในการสร้างโครงสร้างของเว็บเพจ

<!DOCTYPE html>
<html>
<head>
  <title>ตัวอย่างโครงสร้าง HTML</title>
</head>
<body>
  <h1>สวัสดี HTML</h1>
  <p>นี่คือหน้าเว็บตัวอย่าง</p>
</body>
</html>

Introduction

HTML คือภาษาที่ใช้กำหนดโครงสร้างของเว็บเพจ ประกอบด้วยแท็ก (Tag) ต่าง ๆ เพื่อบอกเบราว์เซอร์ว่าข้อมูลแต่ละส่วนคืออะไร เช่น ย่อหน้า รูปภาพ หัวข้อ ตาราง ฯลฯ

<!DOCTYPE html>
<html lang="th">
<head>
  <meta charset="UTF-8">
  <title>แนะนำ HTML</title>
</head>
<body>
  <h2>HTML คืออะไร?</h2>
  <p>HTML ย่อมาจาก HyperText Markup Language...</p>
</body>
</html>

Editors

สามารถเขียน HTML ได้ด้วยโปรแกรมแก้ไขข้อความหลากหลาย เช่น Notepad, Visual Studio Code, Sublime Text, Atom เป็นต้น

Basic

โครงสร้างพื้นฐานของ HTML ประกอบด้วยการประกาศ <!DOCTYPE html> และแท็กหลัก <html>, <head>, <body>

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>หน้าเว็บพื้นฐาน</title>
</head>
<body>
  <h1>สวัสดีโลก!</h1>
  <p>นี่คือหน้าเว็บ HTML แบบพื้นฐาน</p>
</body>
</html>

Elements

Element คือส่วนประกอบใน HTML ที่ประกอบด้วยแท็กเปิด, เนื้อหา, และแท็กปิด เช่น <p>ข้อความ</p>

<h2>หัวข้อ</h2>
<p>ย่อหน้าแรก</p>
<p>ย่อหน้าที่สอง</p>

Attributes

Attribute คือคุณสมบัติที่เพิ่มให้กับแท็ก เช่น id, class, style, href, src ฯลฯ

<h1 id="main-title">นี่คือหัวเรื่องหลัก</h1>
<p class="highlight">ข้อความนี้มีคลาสชื่อ highlight</p>

Headings

แท็กหัวเรื่องใน HTML มีตั้งแต่ <h1> (ใหญ่ที่สุด) ถึง <h6> (เล็กที่สุด) เพื่อจัดลำดับความสำคัญของหัวข้อ

<h1>หัวข้อหลัก</h1>
<h2>หัวข้อรอง</h2>
<h3>หัวข้อย่อย</h3>
<h4>หัวข้อย่อยเพิ่มเติม</h4>

Paragraphs

ย่อหน้าใน HTML ใช้แท็ก <p> เพื่อแบ่งข้อความเป็นบรรทัด/ย่อหน้าอย่างชัดเจน

<p>นี่คือย่อหน้าแรก</p>
<p>นี่คือย่อหน้าที่สอง</p>

Styles

สามารถกำหนดสไตล์แบบอินไลน์ด้วย Attribute style หรือเชื่อมต่อไฟล์ CSS ภายนอก

<p style="color: blue;">ข้อความสีน้ำเงิน</p>
<!-- หรือเชื่อมต่อไฟล์ CSS -->
<link rel="stylesheet" href="styles.css">

Formatting

การจัดรูปแบบข้อความด้วยแท็ก เช่น <b> (ตัวหนา), <i> (ตัวเอียง), <u> (ขีดเส้นใต้), <mark> (ไฮไลท์)

<p>นี่คือ <b>ข้อความตัวหนา</b> และ <i>ข้อความตัวเอียง</i></p>
<p>นี่คือ <mark>ข้อความที่ถูกไฮไลท์</mark></p>

Quotation

การอ้างอิงข้อความด้วย <blockquote> (ย่อหน้าสำหรับการอ้างอิงยาว ๆ) และ <q> (อ้างอิงสั้น ๆ)

<blockquote>
  นี่คือข้อความที่ถูกอ้างอิงแบบยาว
</blockquote>

<p>นี่คือข้อความอ้างอิงสั้น ๆ: <q>Hello</q></p>

Comments

การใส่คอมเมนต์ใน HTML ใช้ <!-- --> เบราว์เซอร์จะไม่แสดงผลคอมเมนต์บนหน้าเว็บ

<!-- นี่คือคอมเมนต์ -->
<p>ข้อความนี้จะแสดงผล</p>

Colors

การกำหนดสีให้กับข้อความหรือพื้นหลัง (ส่วนใหญ่ทำผ่าน CSS) เช่น ใช้ชื่อสี, รหัสสี Hex, หรือ RGB

<p style="color: red;">ข้อความสีแดง</p>
<p style="background-color: #00ff00;">พื้นหลังสีเขียว</p>

HTML Fonts

เดิมเคยมีแท็ก <font> สำหรับกำหนดฟอนต์ แต่ปัจจุบันแนะนำให้ใช้ CSS (เช่น font-family, font-size)

<p style="font-family: Arial; font-size: 18px;">
  ข้อความด้วยฟอนต์ Arial ขนาด 18px
</p>

Links

การสร้างลิงก์ใช้แท็ก <a> โดยกำหนด URL ใน href

<a href="https://www.example.com" target="_blank">
  ไปยัง Example.com
</a>

Images

การแทรกรูปภาพใช้แท็ก <img> และระบุแหล่งรูป (src) รวมถึงข้อความสำรอง (alt)

<img src="picture.jpg" alt="รูปภาพตัวอย่าง" width="200" height="150">

Favicon

การเพิ่มไอคอน (Favicon) บนแท็บเบราว์เซอร์ ใช้แท็ก <link rel="icon" ... > ในส่วน <head>

<head>
  <link rel="icon" type="image/png" href="favicon.png">
</head>

Page Title

กำหนดชื่อหน้าเว็บด้วยแท็ก <title> ภายใน <head>

<head>
  <title>ชื่อหน้าเว็บ</title>
</head>

Tables & Inline

ตารางใน HTML ใช้ <table>, <tr>, <td> และเรียนรู้ความแตกต่าง Block กับ Inline เช่น <span> (inline) และ <div> (block)

<table border="1">
  <tr>
    <th>ชื่อ</th>
    <th>อายุ</th>
  </tr>
  <tr>
    <td>สมชาย</td>
    <td>25</td>
  </tr>
</table>

<div>นี่คือ block element</div>
<span>นี่คือ inline element</span>

Lists

การสร้างรายการแบบไม่มีลำดับ (ul) และมีลำดับ (ol) ใช้ <li> สำหรับแต่ละรายการ

<ul>
  <li>รายการที่ 1</li>
  <li>รายการที่ 2</li>
</ul>

<ol>
  <li>รายการที่ 1</li>
  <li>รายการที่ 2</li>
</ol>

Block & Inline

Block elements (เช่น <div>, <p>) จองพื้นที่เต็มบรรทัด ส่วน Inline elements (เช่น <span>, <a>) จองพื้นที่ตามขนาดเนื้อหา

<div>Block element</div>
ข้อความปกติ <span>Inline element</span> ข้อความปกติ

Div

<div> ใช้สำหรับจัดกลุ่มหรือแบ่งส่วนต่าง ๆ ในหน้า เพื่อความเป็นระเบียบและสะดวกต่อการจัดสไตล์ด้วย CSS

<div>
  <h2>ส่วนหัวข้อใน div</h2>
  <p>เนื้อหาใน div</p>
</div>

Classes

กำหนดคลาสด้วย class attribute เพื่อใช้จัดสไตล์หรืออ้างอิงผ่าน JavaScript ได้

<div class="container">
  <p class="highlight">ข้อความไฮไลท์</p>
</div>

Id

กำหนด id เพื่อระบุองค์ประกอบเพียงหนึ่งเดียวในหน้า (ต้องไม่ซ้ำกัน) เช่น id="main"

<h1 id="page-title">หัวข้อหลัก</h1>
<div id="content">เนื้อหา...</div>

Iframes

<iframe> ใช้ฝังหน้าเว็บหรือวิดีโอจากเว็บอื่น (เช่น YouTube) ลงในหน้าเว็บของเรา

<iframe width="560" height="315"
  src="https://www.youtube.com/embed/XXXXXXXX"
  frameborder="0"
  allowfullscreen>
</iframe>

JavaScript

สามารถฝัง JavaScript ใน HTML เพื่อเพิ่มความโต้ตอบ (Interactive) ให้กับหน้าเว็บ เช่น การตรวจสอบฟอร์มหรือแสดง/ซ่อนเนื้อหา

<script>
  alert('สวัสดี JavaScript!');
</script>

File Paths

การอ้างอิงไฟล์ด้วย Relative Path หรือ Absolute Path เพื่อเชื่อมโยงรูปภาพ ไฟล์ CSS/JS ภายในโปรเจค

<!-- Relative Path -->
<img src="images/pic.jpg" alt="รูปภาพ">

<!-- Absolute Path -->
<img src="https://www.example.com/images/pic.jpg" alt="รูปภาพ">

Head

ส่วน <head> ใช้ใส่ข้อมูลเกี่ยวกับเอกสาร เช่น <title>, <meta>, <link>, <script> (กรณีแทรกไฟล์ JS)

<head>
  <meta charset="UTF-8">
  <title>ตัวอย่าง Head</title>
  <link rel="stylesheet" href="styles.css">
</head>

Layout

การจัดโครงร่างหน้าเว็บ (Layout) มักใช้ <header>, <nav>, <section>, <article>, <aside>, <footer> เพื่อความชัดเจน

<header>ส่วนหัวเว็บ</header>
<nav>เมนูนำทาง</nav>
<section>เนื้อหาหลัก</section>
<aside>เนื้อหาข้างเคียง</aside>
<footer>ส่วนท้ายเว็บ</footer>

Responsive

การทำเว็บให้แสดงผลสวยงามบนหน้าจอหลายขนาด ใช้ meta viewport และ Media Queries ใน CSS

<head>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

<!-- ตัวอย่าง Media Query ใน CSS -->
@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

ComputerCode

แท็กสำหรับแสดงโค้ดหรือข้อความที่เกี่ยวกับคอมพิวเตอร์ เช่น <code>, <pre>, <kbd>, <samp>

<p>ตัวอย่างโค้ด: <code>console.log('Hello')</code></p>

<pre>
function greet() {
  console.log("Hello World");
}
</pre>

Semantics

แท็กเชิงความหมาย (Semantic Tags) เช่น <header>, <footer>, <main>, <article>, <section>, <nav> ช่วยให้โครงสร้างเว็บชัดเจนขึ้น

<article>
  <h2>บทความ</h2>
  <p>เนื้อหาของบทความ</p>
</article>

Style Guide

แนวทางในการเขียนโค้ดให้เป็นระเบียบ เช่น การย่อหน้าโค้ด (Indentation), ตั้งชื่อคลาส/ไอดีให้อ่านง่าย, จัดโครงสร้างไฟล์ เป็นต้น

<!-- ตัวอย่างโค้ดที่มีการย่อหน้าเป็นระเบียบ -->
<div class="content">
  <p>ข้อความ...</p>
  <img src="image.jpg" alt="รูป">
</div>

Entities

HTML Entities ใช้แสดงอักขระพิเศษ เช่น &lt; (<), &gt; (>), &amp; (&)

<p>เขียน &lt;html&gt; แทน <html> เพื่อไม่ให้แท็กทำงาน</p>

Symbols

การแสดงสัญลักษณ์พิเศษ เช่น ©, ®, ™ ผ่าน HTML Entities เช่น &copy;, &reg;, &trade;

<p>สัญลักษณ์ลิขสิทธิ์: &copy;</p>
<p>สัญลักษณ์การค้า: &trade;</p>

Emojis

สามารถใส่อีโมจิในหน้า HTML ได้หากรองรับ Unicode เช่น &#128512; หรือใส่ตรง ๆ (หาก Encoding เป็น UTF-8)

<p>สวัสดีครับ 😀</p>
<p>หรือใส่ตรง: สวัสดีครับ 😀</p>

Charsets

การกำหนดชุดอักขระ (Charset) เช่น UTF-8 เพื่อรองรับภาษาต่าง ๆ และอีโมจิ

<meta charset="UTF-8">

URL Encode

การเข้ารหัสอักขระพิเศษใน URL เช่น การแทนเครื่องหมายวรรคตอนหรือช่องว่าง เพื่อให้ส่งผ่าน URL ได้อย่างถูกต้อง

ตัวอย่างเช่น การแทนช่องว่างด้วย %20:
https://www.example.com/search?q=hello%20world
