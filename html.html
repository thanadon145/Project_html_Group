คู่มือ HTML เบื้องต้น

1. Tag พื้นฐาน

แท็ก HTML คือคำสั่งที่อยู่ในวงเล็บมุม (< >) เพื่อกำหนดโครงสร้างของเอกสาร เช่น <html>, <head>, <body>

<!DOCTYPE html>
<html>
  <head>
    <title>ตัวอย่าง HTML</title>
  </head>
  <body>
    <p>สวัสดีโลก!</p>
  </body>
</html>

2. Attribute

Attribute คือคุณสมบัติเพิ่มเติมในแท็ก HTML เช่น id, class, และ style

<h1 id="main-title">หัวข้อหลัก</h1>
<p class="highlight">ย่อหน้าที่มีการไฮไลท์</p>
<p style="color: blue;">ย่อหน้าสีน้ำเงิน</p>

3. โครงสร้าง HTML พื้นฐาน

โครงสร้างเอกสาร HTML ควรประกอบด้วยการประกาศ DOCTYPE และแบ่งเป็นส่วน <head> และ <body>

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>หน้าเว็บของฉัน</title>
  </head>
  <body>
    <h1>สวัสดี</h1>
    <p>ยินดีต้อนรับสู่เว็บเพจของฉัน!</p>
  </body>
</html>

4. ฟอร์มและปุ่ม

ฟอร์มใช้สำหรับรับข้อมูลจากผู้ใช้ สร้างด้วยแท็ก <form> และองค์ประกอบเช่น <input>, <button>, <label>

<form action="#" method="POST">
  <label for="username">ชื่อผู้ใช้:</label>
  <input type="text" id="username" name="username">
  <br><br>
  <label for="password">รหัสผ่าน:</label>
  <input type="password" id="password" name="password">
  <br><br>
  <button type="submit">ลงทะเบียน</button>
</form>

5. ลิงก์และรูปภาพ

ลิงก์สร้างด้วยแท็ก <a> โดยระบุ URL ใน attribute href ส่วนรูปภาพสร้างด้วยแท็ก <img> โดยต้องระบุ src และ alt

<a href="https://www.example.com" target="_blank">ไปที่ Example.com</a>
<img src="picture.jpg" alt="คำอธิบายรูปภาพ" width="200">

6. ตารางและรายการ

ตารางสร้างด้วยแท็ก <table> พร้อมกับ <tr>, <th>, และ <td> ส่วนรายการสร้างด้วย <ol> (รายการมีลำดับ) หรือ <ul> (รายการไม่มีลำดับ) โดยใช้ <li> สำหรับแต่ละรายการ

<table border="1">
  <tr>
    <th>ชื่อ</th>
    <th>อายุ</th>
  </tr>
  <tr>
    <td>สมชาย</td>
    <td>25</td>
  </tr>
</table>

<ol>
  <li>แอปเปิ้ล</li>
  <li>กล้วย</li>
  <li>เชอร์รี่</li>
</ol>

<ul>
  <li>สุนัข</li>
  <li>แมว</li>
  <li>นก</li>
</ul>
