คู่มือ CSS เบื้องต้น

1. Selector พื้นฐาน

Selectors ใช้เลือกองค์ประกอบที่ต้องการจัดสไตล์ เช่น Element, Class, ID, และ Grouping Selector

p {
  color: blue;
}
.highlight {
  background-color: yellow;
}
#main-title {
  font-size: 24px;
}
h1, h2 {
  font-family: Arial, sans-serif;
}

2. Box Model

Box Model คือแนวคิดการมององค์ประกอบเป็นกล่อง ซึ่งประกอบด้วย Content, Padding, Border และ Margin

.box {
  width: 150px;
  height: 100px;
  padding: 10px;
  margin: 20px;
  border: 2px solid black;
  background-color: lightblue;
}

3. การจัดวาง Layout (Flexbox และ Grid)

Flexbox ใช้สำหรับจัดเรียงองค์ประกอบในแนวเดียว (แนวนอนหรือแนวตั้ง) ส่วน Grid ใช้สำหรับจัดเรียงในรูปแบบแถวและคอลัมน์

Flexbox

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 200px;
}

Grid

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 10px;
}
.grid-item {
  background-color: #ccc;
  padding: 20px;
  text-align: center;
}

4. สีและพื้นหลัง

กำหนดสีข้อความและพื้นหลังด้วย properties color และ background-color

body {
  background-color: lightblue;
}
p {
  color: #FF0000;
  background-color: yellow;
}

5. Typography (Fonts และ Text Alignment)

ปรับแต่งรูปแบบตัวอักษรและการจัดแนวข้อความด้วย properties เช่น font-family, font-size, font-weight, font-style, และ text-align

h1 {
  font-family: Arial, sans-serif;
  font-size: 32px;
  text-align: center;
  font-weight: bold;
}
p {
  font-size: 16px;
  font-style: italic;
  text-align: justify;
}
