CSS Tutorial

CSS HOME

CSS (Cascading Style Sheets) ใช้สำหรับกำหนดหน้าตาและการจัดวางขององค์ประกอบในหน้าเว็บ

/* ตัวอย่างการกำหนดสีพื้นหลังและสีตัวอักษร */
body {
  background-color: #f0f0f0;
  color: #333;
}

CSS Introduction

CSS แยกการจัดรูปแบบ (Presentation) ออกจากโครงสร้าง (Structure) ของเว็บ ทำให้แก้ไขดีไซน์ได้สะดวกและรวดเร็ว

/* ตัวอย่างการเชื่อมต่อไฟล์ CSS ภายนอก */
<head>
  <link rel="stylesheet" href="styles.css">
</head>

CSS Syntax

ไวยากรณ์ CSS ประกอบด้วย selector ตามด้วย {} ที่ภายในระบุ property: value;

selector {
  property: value;
  property2: value2;
}

CSS Selectors

Selector ใช้เลือกองค์ประกอบที่ต้องการจัดสไตล์ เช่น Element Selector, Class Selector, ID Selector

p {
  color: blue; /* Element Selector */
}
.highlight {
  background-color: yellow; /* Class Selector */
}
#main {
  font-size: 20px; /* ID Selector */
}

CSS How To

การใส่ CSS ลงในหน้าเว็บมี 3 วิธีหลัก: Inline, Internal, External

/* Inline Style */
<p style="color: red;">ข้อความสีแดง</p>

/* Internal Style (ภายใน <style> ในหน้า HTML) */
<style>
  p { color: green; }
</style>

/* External Style (ไฟล์ .css ภายนอก) */
<link rel="stylesheet" href="styles.css">

CSS Comments

การคอมเมนต์ใน CSS ใช้ /* ... */ เพื่ออธิบายโค้ดหรือปิดบางส่วนไม่ให้ทำงาน

/* นี่คือคอมเมนต์ */
p {
  color: red; /* คอมเมนต์สั้นๆ */
}

CSS Colors

การกำหนดสีใน CSS สามารถใช้ชื่อสี, รหัสสี Hex, หรือฟังก์ชัน RGB, HSL

p {
  color: #ff0000;     /* สีแดงด้วย Hex */
  background-color: rgb(0, 255, 0); /* สีเขียวด้วย RGB */
}

Backgrounds

คุณสมบัติ background เช่น background-color, background-image, background-repeat

body {
  background-color: #fafafa;
  background-image: url("bg.jpg");
  background-repeat: no-repeat;
  background-size: cover;
}

Borders

การกำหนดเส้นขอบขององค์ประกอบด้วย border (หนา, ชนิดเส้น, สี)

div {
  border: 2px solid red;
}

Margins

กำหนดระยะห่างด้านนอกของกล่องด้วย margin

.box {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 10px;
  margin-left: 20px;
}

Padding

กำหนดระยะห่างด้านในของกล่องด้วย padding

.box {
  padding: 15px;
}

Height/Width

กำหนดความกว้างและความสูงขององค์ประกอบด้วย width และ height

img {
  width: 200px;
  height: auto; /* ปรับอัตโนมัติตามสัดส่วน */
}

Box Model

ทุกองค์ประกอบใน CSS ถือเป็นกล่อง (Box) ที่ประกอบด้วย Content, Padding, Border, Margin

.box {
  width: 100px;
  padding: 10px;
  border: 1px solid #ccc;
  margin: 20px;
}

Outline

คล้ายกับ border แต่ไม่กินพื้นที่ของกล่อง (วาดเส้นรอบนอก)

div {
  outline: 2px dashed blue;
  outline-offset: 5px; /* ระยะห่างจากขอบกล่อง */
}

Text

การจัดการข้อความด้วยคุณสมบัติ color, text-align, text-decoration, text-transform เป็นต้น

p {
  text-align: center;
  text-decoration: underline;
  text-transform: uppercase;
}

Fonts

การกำหนดฟอนต์ เช่น font-family, font-size, font-weight

h1 {
  font-family: Arial, sans-serif;
  font-size: 2em;
  font-weight: bold;
}

Icons

สามารถใช้ Icon Font (เช่น Font Awesome) หรือใช้รูปภาพ/ไฟล์ SVG สำหรับแสดงไอคอน

/* ตัวอย่างการใช้ Font Awesome */
<i class="fa fa-home"></i> Home

Links

การกำหนดสไตล์ลิงก์ในสถานะต่าง ๆ เช่น a:link, a:hover, a:visited, a:active

a:link {
  color: blue;
}
a:hover {
  color: red;
}

Lists

การปรับสไตล์รายการ เช่น การนำ bullet ออก หรือเปลี่ยนรูป bullet

ul {
  list-style-type: none; /* เอา bullet ออก */
}
li {
  padding: 5px 0;
}

Tables

การปรับสไตล์ตาราง เช่น border-collapse, border-spacing, text-align

table {
  border-collapse: collapse;
  width: 100%;
}
th, td {
  border: 1px solid #ccc;
  padding: 8px;
}

Display

กำหนดการแสดงผลขององค์ประกอบ เช่น display: block;, inline;, inline-block;, flex;

div {
  display: inline-block;
  width: 100px;
  height: 100px;
}

Max-width

จำกัดความกว้างสูงสุดขององค์ประกอบเพื่อให้ตอบสนอง (responsive) ได้ดีขึ้น

img {
  max-width: 100%;
  height: auto;
}

Position

กำหนดตำแหน่งขององค์ประกอบ เช่น static, relative, absolute, fixed, sticky

.box {
  position: absolute;
  top: 50px;
  left: 100px;
}

Z-index

ใช้กำหนดลำดับการซ้อนทับขององค์ประกอบที่ถูกจัดตำแหน่ง (positioned elements)

.box1 {
  z-index: 10;
}
.box2 {
  z-index: 20; /* box2 จะอยู่บน box1 */
}

Overflow

กำหนดว่าถ้าขนาดเนื้อหาล้นกล่อง ให้แสดงผลอย่างไร เช่น visible, hidden, scroll, auto

.container {
  width: 200px;
  height: 100px;
  overflow: auto;
}

Float

ใช้จัดองค์ประกอบให้ชิดซ้ายหรือขวา เพื่อวางภาพหรือกล่องขนานกับข้อความ

img {
  float: left;
  margin-right: 10px;
}

Inline-block

การจัดแสดงองค์ประกอบแบบ inline-block ช่วยให้กำหนดขนาดได้ แต่ยังแสดงผลเรียงต่อกันในบรรทัด

.item {
  display: inline-block;
  width: 100px;
  height: 50px;
  background-color: #ccc;
}

Align

การจัดตำแหน่งข้อความหรือองค์ประกอบ เช่น text-align (จัดข้อความ), vertical-align (จัดแนวตั้ง)

p {
  text-align: center;
}
span {
  vertical-align: middle;
}

Combinators

ตัวเลือกที่ใช้ความสัมพันธ์ระหว่างองค์ประกอบ เช่น Descendant (space), Child (>), Sibling (+, ~)

/* Descendant selector */
div p {
  color: blue;
}

/* Child selector */
div > p {
  color: red;
}

Pseudo-class

ใช้เลือกสถานะพิเศษขององค์ประกอบ เช่น :hover, :focus, :active

a:hover {
  color: orange;
}
input:focus {
  border-color: blue;
}

Pseudo-elements

สร้างหรือเลือกส่วนหนึ่งขององค์ประกอบ เช่น ::before, ::after, ::first-line

p::first-line {
  font-weight: bold;
  color: red;
}
p::after {
  content: " (จบ)";
}

Opacity

กำหนดความโปร่งใสขององค์ประกอบ ค่า 0 = โปร่งใสหมด, 1 = ทึบสนิท

.overlay {
  opacity: 0.5;
}

Navigation Bar

การสร้างเมนูนำทาง (Navbar) ด้วย ul, li และสไตล์ เช่น float หรือ flex

nav ul {
  list-style-type: none;
  display: flex;
  background: #333;
}
nav li {
  margin-right: 15px;
}
nav a {
  color: #fff;
  text-decoration: none;
}

Dropdowns

เมนูแบบเลื่อนลง (Dropdown) สามารถใช้ :hover หรือ JavaScript เพื่อควบคุมการแสดงผล

.dropdown:hover .dropdown-content {
  display: block;
}
.dropdown-content {
  display: none;
  position: absolute;
  background: #fff;
}

Image Gallery

จัดรูปภาพเป็นแกลเลอรี อาจใช้ float หรือ flex หรือ grid

.gallery {
  display: flex;
  flex-wrap: wrap;
}
.gallery img {
  width: 200px;
  margin: 5px;
}

Image Sprites

รวมรูปภาพหลาย ๆ รูปไว้ในไฟล์เดียว แล้วใช้ background-position เพื่อแสดงเฉพาะส่วนที่ต้องการ

.icon {
  width: 32px; height: 32px;
  background: url("icons-sprite.png") no-repeat;
}
.icon-home {
  background-position: 0 0;
}
.icon-mail {
  background-position: -32px 0;
}

Attr Selectors

เลือกองค์ประกอบโดยดูจากค่า Attribute เช่น [type="text"], [href^="https"], [class~="highlight"]

input[type="text"] {
  border: 1px solid #ccc;
}
a[href^="https"] {
  color: green;
}

Forms

ปรับสไตล์ฟอร์ม เช่น input, textarea, button, ระยะห่าง และการจัดวาง

form {
  width: 300px;
  margin: 0 auto;
}
input, textarea {
  width: 100%;
  margin-bottom: 10px;
}

Counters

CSS Counters ใช้นับและแสดงตัวเลขอัตโนมัติในรายการหรือส่วนต่าง ๆ ของเนื้อหา

ol {
  counter-reset: section;
}
li {
  counter-increment: section;
}
li::before {
  content: counter(section) ". ";
}

Website Layout

การจัดโครงร่างเว็บ (Layout) นิยมใช้ flexbox หรือ grid ช่วยให้จัดวางองค์ประกอบได้สะดวก

.container {
  display: flex;
}
.sidebar {
  width: 200px;
}
.content {
  flex: 1;
}

Units

หน่วยที่ใช้ใน CSS เช่น px, em, rem, %, vw, vh

p {
  font-size: 16px; /* px */
}
.container {
  width: 80%; /* % */
}

Specificity

ความสำคัญของ Selector ในการตัดสินว่า CSS ตัวไหนมีผล เช่น Inline > ID > Class > Element

/* ตัวอย่าง */
#box { color: red; }     /* ID */
.box { color: blue; }    /* Class */
div { color: green; }    /* Element */

/* #box ชนะ .box และ div */

!important

ใช้ในกรณีต้องการให้ property นั้นมีความสำคัญสูงสุด แต่ควรใช้อย่างระมัดระวัง

p {
  color: blue !important;
}

Math Functions

CSS มีฟังก์ชันคำนวณ เช่น calc(), min(), max(), clamp() เพื่อช่วยปรับขนาด/ระยะห่างตามเงื่อนไข

.box {
  width: calc(100% - 50px);
  font-size: clamp(16px, 2vw, 24px);
}
